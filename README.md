# AI Google Ads Campaign Automation Agent

This project demonstrates an AI agent that can automatically generate and prepare Google Ads campaigns using AI.

The system takes a product/service description and automatically generates:

- Google Ads headlines
- Ad descriptions
- Keyword suggestions
- Campaign structure

This project demonstrates how AI agents can automate marketing workflows.

## Features

- AI powered ad copy generation
- Keyword generation
- Campaign JSON structure
- API ready output
- Automation ready design

## Tech Stack

- Python
- OpenAI / Claude API
- FastAPI
- LangChain
- JSON workflow

## Example Input

Business:
Personal Injury Law Firm

Location:
California

Service:
Car Accident Lawyer

## Example Output

Headlines:
- Experienced Car Accident Lawyer
- Get Legal Help After a Crash
- Free Case Evaluation Today

Keywords:
- car accident lawyer

- accident legal help
